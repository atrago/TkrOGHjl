## 核心代码

以下是本项目中的一段核心代码，展示了如何使用Spring Boot框架实现一个简单的RESTful接口：

```java
@RestController
@RequestMapping("/api")
public class HelloWorldController {

    @GetMapping("/hello")
    public String hello() {
        return "Hello, World!";
    }
}
```

这段代码定义了一个名为HelloWorldController的控制器类，其中包含一个GET类型的映射，访问路径为/api/hello。当用户访问该路径时，将会返回"Hello, World!"字符串。
